== Description ==

同步发表 WordPress 博客日志到乐收,初次安装必须设置后才能使用。

Related Links:

* <a href="http://plugins.191998.net/leshou" title="乐收" target="_blank">乐收</a>

Version 1.0支持功能

1.支持将Wordpress中标题和链接发布到乐收;
2.此插件仅支持乐收vip用户；

== Installation ==

1. 上传 `leshou.php` 到 `/wp-content/plugins/leshou/` 目录

2. 在Wordpress后台控制面板"插件(Plugins)"菜单下激活乐收插件

3. 在Wordpress后台控制面板"配置(Settings)->乐收"菜单下设置插件的必须信息。（只有经过设置，插件才能正常使用）
